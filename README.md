<p style='margin-bottom:7.9pt'><b><span style='font-family:"Cambria",serif'>Porting

<p style='margin-bottom:7.9pt'><span style='font-family:"Cambria",serif'>Before
we start creating <span class=SpellE>Nuget</span> packages for .net framework
projects for different platforms. <span class=SpellE>Letâ</span>€™s first

<p style='margin-bottom:7.9pt'><span style='font-family:"Cambria",serif'>As we
all know .Net core is new web Framework, Microsoft has launched which is
platform independent. .Net Core Framework is lite weight compared to .Net
Framework. Apps developed in .Net core can be deployed and run on other
platforms like <span class=SpellE>MacOS</span>, Linux, <span class=GramE>Windows</span>
etc. </span></p>

<p style='margin-bottom:7.9pt'><span style='font-family:"Cambria",serif'>But In
our scenario, we are <span class=GramE>developing <span

will used as a new API end-point to use <b style='mso-bidi-font-weight:normal'>riskware</b>
services. As we are building this project in .Net core, we want our existing
services to support this project which are created using .Net Framework. In
order to achieve this we need to convert our .Net Framework <span class=SpellE>dll’s</span>
to .Net Core. <o:p></o:p></span></p>

<p style='margin-bottom:7.9pt'><i style='mso-bidi-font-style:normal'><span
style='font-family:"Cambria",serif'>Note: As we want to use the existing
riskware services in the new RAPI Project which is in .Net Core, we need to
make sure that the legacy application should be working and fully functional.<o:p></o:p></span></i></p>

<p style='margin-bottom:7.9pt'><span style='font-family:"Cambria",serif'>In
this document, I will run you through the process of converting .Net Framework
projects into .Net Core. First <span class=SpellE><span class=GramE>lets</span></span>
start the process, by creating a <span class=SpellE>Nuget</span> Package. But before
we do that we need to check the dependency on the project. If there is a
dependency on that project, then you need to first create the package for that
project. <o:p></o:p></span></p>
